# AIAS Route Overview & Purpose Map

This document maps the key front-end routes for the AIAS consultancy website, identifying their purpose, target persona, and conversion goals.

## Core Marketing / Consultancy Routes

| Route | Purpose | Target Persona | Primary Conversion Goal |
|-------|---------|----------------|-------------------------|
| `/` | Homepage. High-level value prop, trust building, navigation hub. | Executive / Founder | "Book a Strategy Call" / "View Services" |
| `/services` | Detailed breakdown of offerings (Strategy, Implementation, Advisory). | Ops / Tech Lead | "Book a Consultation" |
| `/case-studies` | Social proof, demonstrating ROI and success stories. | Executive / Founder | "See How We Can Help You" (Lead Gen) |
| `/about` | Company mission, team expertise, "Why Us". | Founder / Ops | Trust building -> Contact |
| `/pricing` | Transparency on engagement models (Retainer, Project, Hourly). | Decision Maker | "Get a Quote" / "Choose Plan" |
| `/demo` | Primary contact/conversion point. Calendar booking or form. | High Intent Lead | **Book Meeting** |
| `/blog` | Thought leadership, SEO traffic entry point. | Broad (Research phase) | Newsletter Signup / Internal Link to Services |
| `/features` | Product/Platform capabilities (Hybrid consultancy/SaaS model). | Tech Lead / Ops | "Start Trial" / "Request Demo" |
| `/integrations` | Technical proof of capability (e.g., Shopify, Wave). | Tech Lead | "Check Compatibility" |
| `/why-canadian` | Niche positioning (Data sovereignty, local expertise). | Canadian Execs | Trust building |
